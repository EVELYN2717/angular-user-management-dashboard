# 🚀 User Management Dashboard (Angular)

Aplicación web desarrollada con Angular que permite gestionar usuarios mediante un CRUD completo.

## ✨ Funcionalidades
- Listado de usuarios (API REST)
- Crear usuario
- Editar usuario
- Eliminar usuario
- Validaciones con Reactive Forms
- Feedback visual con Angular Material

## 🛠️ Tecnologías
- Angular
- TypeScript
- Angular Material
- RxJS
- REST API

## 🎯 Objetivo
Proyecto enfocado en demostrar habilidades en desarrollo frontend, arquitectura modular y buenas prácticas.

## ▶️ Ejecución

```bash
npm install
ng serve
